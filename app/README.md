# PFVue App

This directory is a real Vite + Vue 3 application workspace inside the PFVue practice repository.

Role of this folder:

- use it after you finish the lesson-first practice layers
- treat it as the bridge from exercises to a formal Vue project
- iterate here when you want to practice real folder structure and app evolution

What it includes:

- Vite-based project structure
- Vue Router
- Pinia stores
- mock API modules
- composables
- page and component separation

Suggested commands:

```bash
npm install
npm run dev
npm run build
```

Suggested learning path:

1. read `src/router/index.js`
2. read `src/stores/tasks.js`
3. read `src/views/DashboardView.vue`
4. modify the task workflow and add new pages later
