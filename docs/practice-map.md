# PFVue Practice Map

This file is the fastest way to understand where to practice what.

## 1. If You Are A Beginner

Start here:

1. `foundations/README.md`
2. `intermediate/README.md`
3. `vue3_core/README.md`

Goal:

- learn HTML, CSS, JavaScript, DOM, async basics, and Vue 3 essentials

## 2. If You Already Know Frontend Basics

Start here:

1. `vue3_core/README.md`
2. `advanced/README.md`
3. `projects/README.md`

Goal:

- practice Vue 3 component thinking and move into real project workflows

## 3. If You Want Real Project Practice

Start here:

1. `projects/01_component_playground`
2. `projects/02_admin_dashboard`
3. `projects/03_real_world_app`

Goal:

- move from isolated lessons into page-level and workflow-level implementation

## 4. If You Want Formal Vue Project Structure

Start here:

1. `app/README.md`
2. `app/src/main.js`
3. `app/src/router/index.js`
4. `app/src/stores/tasks.js`
5. `app/src/views/DashboardView.vue`

Goal:

- practice Vite, Vue Router, Pinia, composables, and source code organization
