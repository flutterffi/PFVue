# 03 Real World App

This project is a more complete Vue 3 practice page with a simple app flow.

What it includes:

- a lightweight sign-in gate
- summary metrics
- task search and status filtering
- a list-detail-edit workflow
- inline task editing with shared state updates

Use it to practice:

- page flow after authentication
- coordinating list state with a detail editor
- editing shared data and reflecting changes across the page
